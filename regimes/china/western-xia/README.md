# 西夏 · 多民族州郡制 (Western Xia)

> 1038-1227 AD · Multi-ethnic Prefecture System

## 简介

西夏（1038-1227）由党项族李元昊建立，定都兴庆府（今宁夏银川），控制河西走廊战略要地。

西夏的治理特色是军事优先、佛教辅政、文化自主：
- **军事优先**：党项族以武立国，枢密使掌军事大权
- **佛教辅政**：国师参与国政，佛教深刻影响决策
- **文化自主**：创制西夏文字，坚持独立的文化认同
- **多族共治**：蕃汉大学士负责协调党项、汉、吐蕃、回鹘等各族事务

## 历史背景

党项族原为西北羌系民族，经数代经营，李元昊于1038年正式称帝建国。
西夏地处宋、辽、金之间的战略要冲，凭借河西走廊的地利和卓越的外交手腕，
在三大强国的夹缝中存续近两百年。

**核心思想：自主生存**
- 创制西夏文字（仿汉字结构，独立造字），彰显国家独立
- 尊崇佛教，大量翻译佛经，以佛法教化国民
- 灵活外交，在宋辽金之间左右逢源
- 军事立国，全民皆兵，以少胜多

## Agent 配置

本政体包含 5 个 Agent：

| Agent | 角色 | 职责 |
|---|---|---|
| 皇帝 | 最高决策 | 总揽军政、战略决策、全局统帅 |
| 中书令 | 行政执行 | 总理政务、政令起草执行、官员管理 |
| 枢密使 | 军事技术 | 军事机密、兵马调度、技术架构 |
| 国师 | 佛教/知识 | 佛法顾问、知识库管理、文化政策 |
| 蕃汉大学士 | 民族/翻译 | 蕃汉翻译、多民族协调、文化沟通 |

## 使用方式

```bash
./scripts/switch-regime.sh china/western-xia
```

## 编排模式

**Centralized Hierarchy (中央集权)**: 皇帝（main agent）总揽一切，中书令管行政，枢密使管军事，
国师提供智慧参考，蕃汉大学士负责跨文化协调。适合小团队、资源有限但需要高效决策的项目。
