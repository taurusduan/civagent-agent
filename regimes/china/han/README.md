# 汉朝 · 三公九卿制 (Han Dynasty)

> 公元前206年-公元220年 · Three Lords Nine Ministers System

## 简介

汉朝（前206-220）是继秦朝之后的第二个大一统王朝，也是中国历史上第一个长期稳定的帝国，
延续四百余年（含西汉、东汉）。汉朝奠定了"汉族"、"汉字"、"汉语"的文化根基，
其制度深远影响了此后两千年的中国政治。

### 核心制度：外儒内法

汉初沿用秦制，经汉武帝"罢黜百家，独尊儒术"，形成了独特的"外儒内法"治理模式：
- **外儒**：以儒家仁义礼智信为官方意识形态，强调德治、教化
- **内法**：以法家制度为实际运作基础，强调规则、考核、赏罚

这种模式使得汉朝既有秦朝的行政效率，又避免了秦朝因纯法家暴政而速亡的命运。

### 三公九卿

- **三公**：丞相（总理政务）、太尉（掌军事）、御史大夫（掌监察）
- **九卿**：太常、光禄勋、卫尉、太仆、廷尉、大鸿胪、宗正、大司农、少府

### 察举制

汉朝通过察举制选拔人才：地方官推荐品德优良、才能出众的人士入朝为官。
这是科举制的前身，强调"唯才是举"的用人原则。

## 编排模式

**Centralized Hierarchy (中央集权)**: 丞相总理万机，统一调度九卿。
以儒家德治为指导思想，以法家制度为运作基础。兼容并蓄，德法兼施。
