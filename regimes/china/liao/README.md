# 辽朝 · 南北面官制 (Liao Dynasty)

> 907-1125 AD · Dual Administration System

## 简介

辽朝（907-1125）由契丹族耶律阿保机建立，是中国历史上第一个成功实行双重行政制度的政权。

辽朝的核心创新是南北面官制：
- **北面官**：以契丹传统制度管理游牧民族事务（军事、部落、畜牧）
- **南面官**：以汉族官制管理汉地农耕事务（行政、税赋、司法）

两套体系并行不悖，各司其职，由皇帝（大汗/天子双重身份）统一裁决。

## 历史背景

契丹族原为北方草原游牧民族，耶律阿保机统一契丹八部后建国。
灭渤海国、占据燕云十六州后，面对辖区内游牧与农耕两种截然不同的社会形态，
创造性地采用了「因俗而治」的双轨制度。

**核心思想：因俗而治**
- 不强行统一制度，尊重各族生活方式
- 游牧区用契丹旧制，农耕区用汉族官制
- 这一制度影响了后来的金、元等朝代

本政体包含 6 个 Agent：

| 北院大王 | 北面官之首 | 总调度、接收汗令、协调南北两面 |
| 南院大王 | 南面官之首 | 汉地行政、税赋管理、民政事务 |
| 北面林牙 | 契丹军事 | 军事部署、技术架构、系统安全 |
| 南面宰相 | 汉制行政 | 政令起草、文书管理、流程规范 |
| 夷离堇 | 部落管理 | 部落事务、资源调配、后勤保障 |
| 敌烈麻都 | 监察审查 | 巡视百官、代码审查、质量监控 |

## 编排模式

**Dual Power (双轨并行)**: 北面官和南面官各管一摊，互不干涉，由北院大王（main agent）统一协调。
适合需要管理多种技术栈、多语言代码库、或前后端分离架构的项目。
