# 五代十国 (Five Dynasties and Ten Kingdoms)

> 907-960 AD · Fragmented Warlord System

## 简介

五代十国（907-960）是唐朝灭亡到北宋统一之间的大分裂时期，持续约53年。

这一时期的核心特征：
- **中央虚弱**：朝廷名义上是最高权力中心，但实际控制力极为有限
- **军阀割据**：各节度使拥有独立的军队、财政和行政权，形同独立王国
- **政权频更**：中原五朝更迭，武将篡位成为常态
- **竞争激烈**：各方势力为争夺资源和地盘激烈竞争

这种「联邦自治」模式适合模拟多团队竞争、去中心化协作的场景。

## 历史背景

唐末藩镇割据日益严重，安史之乱后节度使权力膨胀，最终导致唐朝灭亡。
朱温篡唐建后梁，此后后唐、后晋、后汉、后周相继更替，每朝仅存数年到十余年。
南方十国相对稳定，经济文化反而有所发展。最终后周大将赵匡胤陈桥兵变，建立北宋，
结束分裂。五代十国的教训是：过度分权导致内耗，但竞争也催生了创新。

## Agent 配置

本政体包含 5 个 Agent：

| Agent | 角色 | 职责 |
|---|---|---|
| 朝廷 | 名义中央 | 发布政令、名义协调、象征性统筹 |
| 节度使A | 军阀/团队A | 独立运作、擅长特定领域 |
| 节度使B | 军阀/团队B | 独立运作、与他方竞争 |
| 节度使C | 军阀/团队C | 独立运作、各有专长 |
| 掌书记 | 文书/协调 | 信息传递、记录决策、协调冲突 |

## 使用方式

```bash
./scripts/switch-regime.sh china/five-dynasties
```

## 编排模式

**Federated Autonomy (联邦自治)**: 各节度使（团队）高度自治，独立决策，独立执行。
朝廷仅做名义协调。适合需要多方案竞争、A/B 测试、创意竞标的场景。
