# 清朝 · 军机处+满汉合作 (Qing Dynasty)

> 1644-1912 AD · Grand Council + Manchu-Han Cooperation

## 简介

清朝（1644-1912）是中国历史上最后一个封建王朝，也是中国疆域最辽阔的时期之一。
清朝在继承明制的基础上，创设了军机处这一高效决策机构，
并推行满汉合作的民族政策，建立了独特的治理体系。

### 核心制度：军机处

军机处设立于雍正七年（1729），最初是为处理西北军务的临时机构，
后逐渐成为清朝的核心决策中枢，取代内阁成为真正的权力中心。

军机处的特点：
- **精英小组**：军机大臣通常仅3-6人，由皇帝亲自选任
- **直接承旨**：不经内阁等中间环节，直接承接皇帝旨意
- **快速高效**：日清日结，不得积压。"廷寄"制度保证政令快速下达
- **绝对保密**：军机处办事极重保密，"军机"二字即含机密之意

### 满汉合作

清朝推行满汉合作政策：
- 六部设满汉双尚书，满人主导、汉人辅助
- 科举制继续实行，汉族士人可通过考试入仕
- 八旗制度与科举制并行，各取所长

### 督抚制度

清朝地方实行总督巡抚制：
- **总督**：辖一至三省，管军政大权
- **巡抚**：辖一省，管民政
- 总督巡抚直接向皇帝奏报，不受内阁或军机处管辖

## 编排模式

**Centralized Hierarchy (中央集权)**: 军机处精英小组式决策，
皇帝旨意直达军机处，军机处快速分派执行。决策链最短，响应最快。
类似于现代创业公司的核心决策团队（C-Suite）直接拍板的模式。

## 适用场景

- 需要快速决策和执行的紧迫项目（军机处模式天然适合快节奏）
- 核心团队精干、需要高效协作的小团队
- 有地方/区域化部署需求的项目（总督模式）
- 需要独立监察体系的高合规要求项目
