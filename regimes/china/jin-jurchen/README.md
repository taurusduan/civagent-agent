# 金朝 · 女真汉分治 (Jin Dynasty - Jurchen)

> 1115-1234 AD · Jurchen-Han Divided Governance

## 简介

金朝（1115-1234）由女真族完颜阿骨打建立，先灭辽朝，再灭北宋，统治中国北方百余年。

金朝的治理特色是女真军事制度与汉族文治制度的融合：
- **猛安谋克制**：女真核心军事制度，千户为猛安、百户为谋克，兵民合一
- **尚书省六部**：采用汉制行政体系处理日常政务
- **都元帅府**：最高军政统帅机构，统率文武两班

## 历史背景

女真族原居东北白山黑水之间，完颜阿骨打以2500铁骑起兵反辽，十年间灭辽灭宋。
金朝以「女真不满万，满万不可敌」著称，战斗力极强。

入主中原后，金朝面临如何治理广大汉族地区的问题。
采取了逐步汉化的策略：保留猛安谋克制管理女真人，同时设立尚书省六部管理汉地政务。

**核心思想：尚武务实**
- 猛安谋克兵民合一，平时耕牧、战时出征
- 以军功论赏，赏罚分明
- 在保持女真传统的同时务实地吸收汉制

## Agent 配置

本政体包含 6 个 Agent：

| Agent | 角色 | 职责 |
|---|---|---|
| 都元帅 | 最高统帅 | 总揽军政、战略决策、全局调度 |
| 尚书省 | 汉制行政 | 政令执行、行政管理、民政事务 |
| 猛安谋克 | 女真军事 | 兵民合一、技术架构、系统安全 |
| 御史台 | 监察审查 | 纠劾百官、代码审查、质量监控 |
| 翰林学士 | 文书文档 | 诏令起草、文档编写、知识管理 |
| 提刑司 | 司法合规 | 法务合规、安全审计、风险管控 |

## 使用方式

```bash
./scripts/switch-regime.sh china/jin-jurchen
```

## 编排模式

**Dual Power (双轨并行)**: 都元帅统率全局，猛安谋克管军事/技术，尚书省管行政/文治。
适合需要兼顾快速开发与稳定运营、或技术团队与业务团队并行协作的项目。
