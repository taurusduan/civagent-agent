# 中华民国 · 五权分立 (Republic of China)

> 1912-1949 AD · Five-Power Constitution

## 简介

中华民国（1912-1949）是亚洲第一个民主共和国，由孙中山先生领导辛亥革命推翻清朝后建立。

五权宪法是孙中山的独创贡献：
- **行政院**：最高行政机关，执行政策
- **立法院**：制定法律法规
- **司法院**：解释法律、裁判争议
- **考试院**：源自中国科举传统，掌管公务员考试与铨叙
- **监察院**：源自中国御史制度，弹劾纠举、审计监察

在西方三权（行政、立法、司法）基础上增加考试权和监察权，是中国传统与现代民主的创造性融合。

## 历史背景

1911年辛亥革命推翻清朝，结束两千年帝制。孙中山就任临时大总统，提出三民主义
（民族、民权、民生）和五权宪法。此后经历北洋政府、国民革命、抗日战争等历程。
1946年制宪国民大会通过《中华民国宪法》，正式确立五权分立体制。
五权宪法体现了孙中山「以中国固有之考试监察二权，补西方三权之不足」的政治智慧。

本政体包含 7 个 Agent：

| 总统 | 国家元首 | 统筹协调五院、发布政令 |
| 行政院 | 院长 | 行政执行、政策推进、日常运营 |
| 立法院 | 院长 | 立法规则、标准制定、流程规范 |
| 司法院 | 院长 | 司法合规、争议仲裁、法律解释 |
| 考试院 | 院长 | 考核质保、测试验证、人才评估 |
| 监察院 | 院长 | 监察审查、代码审查、安全审计 |
| 国民大会 | 主席 | 代表需求、用户反馈、民意收集 |

## 编排模式

**Checks & Balances (制衡)**: 五院分权制衡，各司其职。行政执行、立法规范、司法裁决、
考试把关、监察监督，形成完整的治理闭环。适合需要严格质量保证和多层审查的项目。
