# 太平天国 · 神权专制 (Taiping Heavenly Kingdom)

> 1851-1864 AD · Theocratic Autocracy

## 简介

太平天国（1851-1864）是中国历史上规模最大的农民起义政权，由洪秀全创立。

太平天国的制度特色：
- **神权政治**：天王自称天父之子，以拜上帝教为立国根基，宗教与政治合一
- **绝对集权**：天王拥有至高无上的权威，一切决策不可质疑
- **五王分治**：东、西、南、北、翼五王分掌军政各领域，东王权力最大
- **平均主义**：天朝田亩制度主张土地平均分配，资源共享
- **狂热执行**：以宗教信仰激发极强的执行力和战斗力

## 历史背景

洪秀全屡试不中，受基督教启发创立拜上帝教，在广西金田村起义。
太平军一路北上，攻克南京并定都「天京」，建立政权与清朝对峙长达十余年。
早期封东王杨秀清、西王萧朝贵、南王冯云山、北王韦昌辉、翼王石达开五王。
杨秀清权势日盛，最终引发天京事变（北王诛杀东王），内部自相残杀，元气大伤。
翼王石达开负气出走，太平天国由盛转衰，最终被湘军攻灭。

## 编排模式

**Theocratic (神权专制)**: 天王拥有绝对权威，东王代为执行调度。
宗教狂热驱动极强执行力，等级森严不可僭越。适合需要强力集中领导和快速执行的紧急项目。
