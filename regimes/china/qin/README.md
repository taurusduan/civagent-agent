# 秦朝 · 三公九卿制 (Qin Dynasty)

> 公元前221年-公元前206年 · Commandery-County System with Three Lords Nine Ministers

## 简介

秦朝（前221年-前206年）是中国历史上第一个统一的中央集权制帝国。秦始皇嬴政统一六国后，
废除了延续千年的分封制，推行郡县制，建立了以三公九卿为核心的中央行政体系。

这是一个崇尚法家思想的政权：以法治国、令行禁止、赏罚分明。

## 历史背景

秦国以商鞅变法崛起，凭借严格的法治制度和高效的行政体系，最终扫灭六国。
统一后推行「书同文、车同轨、统一度量衡」，奠定了中国两千年大一统格局的基础。

**核心思想：法家**
- 韩非子：「法、术、势」三位一体
- 商鞅：「以法为教，以吏为师」
- 李斯：制度设计的实际执行者

## 使用方式

```bash
./scripts/switch-regime.sh china/qin
```

## 编排模式

**Centralized Hierarchy (中央集权)**: 丞相（main agent）统管一切，所有其他 Agent 都是其 subagent。
命令自上而下传达，结果逐级上报。适合需要快速决策和执行的场景。
