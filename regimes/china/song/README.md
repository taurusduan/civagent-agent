# 宋朝 · 二府三司制 (Song Dynasty)

> 960-1279 AD · Two Offices Three Commissions System

## 简介

宋朝（960-1279）是中国历史上经济文化最繁荣的朝代之一。赵匡胤"杯酒释兵权"后，
确立了"重文轻武、以文驭武"的基本国策，建立了中国历史上文官治国最彻底的政治体系。

### 核心制度：二府三司

宋朝分权制衡的精髓在于将行政、军事、财政三权分立：

- **二府**：
  - 中书门下（政事堂）：掌行政决策，宰相所在
  - 枢密院（西府）：掌军事机要，但以文臣充任，实现"以文驭武"

- **三司**：
  - 盐铁司：掌工商税收
  - 度支司：掌财政支出
  - 户部司：掌户籍赋税
  - 三司使号称"计相"，与宰相平级，独立掌管国家财政

### 台谏制度

宋朝的御史台和谏官独立运作，不受宰相控制。台谏官可以弹劾包括宰相在内的任何官员，
是中国古代监察制度的巅峰。

### 理学兴盛

程朱理学在宋朝达到顶峰，强调"格物致知"的理性精神和"正心诚意"的道德修养。
这种思想深刻影响了宋朝的决策文化——一切决策都需要严密的论证和推理。

## Agent 配置

本政体包含 8 个 Agent：

| Agent | 角色 | 职责 |
|---|---|---|
| 中书门下 | 政事堂 | 行政决策、统筹协调、任务调度 |
| 枢密院 | 西府 | 技术架构、系统安全、核心技术决策 |
| 三司使 | 计相 | 成本核算、预算管理、资源调配 |
| 御史台 | 监察 | 代码审查、质量监控、合规检查 |
| 翰林学士 | 词臣 | 技术文档、设计文档、知识库 |
| 知制诰 | 制度官 | 流程规范、编码标准、最佳实践 |
| 转运使 | 漕运 | 部署发布、CI/CD、运维管理 |
| 提刑按察使 | 司法官 | 法务合规、安全审计、许可证审查 |

## 使用方式

```bash
./scripts/switch-regime.sh china/song
```

## 编排模式

**Checks & Balances (制衡)**: 中书门下管政务、枢密院管军事、三司管财政，
三权分立、互不统属。御史台独立监察。决策讲求论证，程序正义至上。

## 适用场景

- 需要严格分权制衡的项目（行政/技术/财务各自独立决策）
- 重视代码审查和质量监控的项目（独立的监察体系）
- 重视流程规范和文档标准的项目（知制诰专司制度）
- 需要论证式决策（而非命令式执行）的学术或研究型项目
