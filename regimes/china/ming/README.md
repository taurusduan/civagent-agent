# 明朝 · 内阁制+司礼监 (Ming Dynasty)

> 1368-1644 AD · Grand Secretariat + Directorate of Ceremonial

## 简介

明朝（1368-1644）是中国历史上最后一个由汉族建立的大一统王朝。
朱元璋废除丞相制度，皇帝直接统辖六部，皇权空前强化。
但随着政务繁重，逐渐形成了内阁票拟、司礼监批红的独特双轨决策体系。

### 核心制度：票拟与批红

明朝政治的独特之处在于"票拟批红"的双轨制：

- **内阁票拟**：内阁大学士（尤其是首辅）审阅各类奏章，在附纸上用墨笔写出
  处理意见（"票拟"），供皇帝参考
- **司礼监批红**：司礼监掌印太监用朱笔批示（"批红"），代行皇帝的审批权

这一制度形成了文臣（内阁）与宦官（司礼监）的微妙平衡。张居正时期，
首辅权力达到巅峰；魏忠贤时期，司礼监权力又压过内阁。

### 六部直属

明朝六部（吏户礼兵刑工）直接对皇帝负责，但实际运作中受内阁和司礼监
双重影响。六部各有明确分工，互不统属。

### 心学崛起

王阳明的"知行合一"和"致良知"在明代中后期广泛传播，
强调实践与认知的统一，成为明代政治哲学的重要流派。

## 编排模式

**Dual Power (双轨)**: 内阁票拟提出方案，司礼监批红审批确认。
重大决策必须经过双轨确认，任何一方都不能单独做最终决定。
类似于现代的 PR 审查机制——开发者提交代码（票拟），审查者批准合并（批红）。

## 适用场景

- 需要双重审批（提案+审核）的高风险项目
- 需要制衡机制防止任何一方独断的团队
- 类似 PR 审查流程的软件开发项目
- 需要文官系统（策划）和执行系统（落地）分离的项目
