# 隋朝 · 三省六部制（初创） (Sui Dynasty)

> 581-618 AD · Three Departments Six Ministries (Proto)

## 简介

隋朝（581-618）是中国历史上承上启下的关键王朝。隋文帝杨坚结束了南北朝近三百年的分裂局面，
建立了统一的中央集权国家。

隋朝的制度创新影响深远：
- **三省六部制**：内史省起草、门下省审核、尚书省执行，开创了决策权、审核权、执行权分立的架构
- **科举制度**：以考试选拔人才，打破门阀士族对仕途的垄断
- **均田制**：合理分配土地资源
- **大运河**：贯通南北的超级工程，展现集中力量办大事的能力

隋朝虽仅存37年（二世而亡），但其制度设计为唐朝的繁荣奠定了基础。

## 历史背景

隋文帝杨坚以北周外戚身份篡位建隋，随后灭陈统一天下。他励精图治，创设三省六部制和科举制，
被称为「开皇之治」。其子隋炀帝杨广修大运河、建东都洛阳、三征高句丽，劳民伤财，
引发全国起义，隋朝灭亡。大运河虽让隋亡，却惠及后世千年。

| 尚书省 | 尚书令 | 分派执行、统筹协调 |
| 吏部 | 尚书 | 人事管理、科举选才、项目管理 |
| 户部 | 尚书 | 财政运营、预算管控、资源调配 |
| 兵部 | 尚书 | 军事技术、软件工程、架构设计 |
| 工部 | 尚书 | 工程运维、DevOps、基础设施 |

## 编排模式

**Checks & Balances (制衡)**: 三省分权制衡，内史起草、门下审核、尚书执行。
初创阶段注重效率，审核流程相对精简但不可省略。大工程精神鼓励承担复杂项目。
