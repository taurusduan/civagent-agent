# 晋朝 · 九品中正制 (Jin Dynasty)

> 266-420 AD · Nine-Rank System

## 简介

晋朝（266年-420年）分为西晋和东晋两个时期。这一时代的政治核心是门阀士族政治与九品中正制。

九品中正制的运作方式：
- **中正品评**：由中正官按家世、行状、才能将人物评定为九等
- **据品授官**：根据品级高低授予相应官职
- **门阀垄断**：实践中演变为「上品无寒门，下品无士族」

晋朝的另一特征是弱中央强地方——中央朝廷权威有限，地方都督、刺史各据一方，拥兵自重。文化上清谈玄学盛行，魏晋名士追求个性解放与精神超脱。

## Agent 配置

本政体包含 6 个 Agent：

| Agent | 角色 | 职责 |
|---|---|---|
| 丞相 | 中央协调 | 斡旋各方、全局调和、争端调解 |
| 中正 | 品评审查 | 代码审查、质量分级、绩效评估 |
| 尚书令 | 行政执行 | 政令推行、任务分配、日常管理 |
| 中书监 | 起草规划 | 方案设计、需求分析、文档编撰 |
| 都督 | 军事技术 | 代码开发、架构设计、技术攻关 |
| 刺史 | 地方部署 | 部署运维、环境管理、地方事务 |

## 编排模式

**Federated Autonomy (联邦自治)**: 丞相作为弱中央进行协调，各Agent（尤其是都督、刺史）在各自领域享有较大自主权。中正官独立进行品评审查，不受其他势力干预。适合需要强审查机制但执行分散的场景。

## 历史特色

- **九品品评**: 严格的分级审查机制，所有成果都要经过品级评定
- **门阀政治**: 资深势力掌握核心资源与话语权
- **清谈之风**: 鼓励从哲学高度审视问题本质
- **弱中央**: 协调为主，不强制干预
