# 周朝 · 宗法分封制 (Zhou Dynasty)

> c. 1046-256 BC · Patriarchal Enfeoffment

## 简介

周朝（约公元前1046年-前256年）是中国历史上存续最久的朝代，历时近800年。周朝确立了影响中国两千年的宗法分封制度和礼乐文明。

宗法分封制的核心特征：
- **天子共主**：天子是名义上的天下共主，起象征性协调作用
- **诸侯自治**：各诸侯国在封地内享有高度自治权
- **礼乐秩序**：以周礼规范一切行为，维系社会秩序
- **嫡长继承**：以嫡长子继承制确保权力传承的稳定

## 编排模式

**Federated Autonomy (联邦自治)**: 天子作为象征性共主进行全局协调，诸侯（独立团队）在各自封国内享有高度自治权。适合多个独立项目并行推进的场景，各团队自主决策、定期汇报。

## 历史特色

- **礼乐治国**: 以周礼规范一切流程，秩序即礼法
- **分封自治**: 各团队高度自治，天子不轻易干预
- **宗法传承**: 严格的主从关系与继承秩序
- **会盟机制**: 跨团队协调通过「会盟」完成
