# 南北朝 · 门阀士族制 (Northern and Southern Dynasties)

> 420-589 AD · Aristocratic Clan System

## 简介

南北朝（420年-589年）是中国历史上南北对峙的分裂时期，最终由隋朝统一。

这一时期的核心特征：
- **南北分治**：南朝重文，北朝尚武，各有所长
- **门阀把政**：门阀士族掌握核心政治资源，「旧时王谢堂前燕，飞入寻常百姓家」
- **民族融合**：北朝鲜卑等族逐步汉化，促进文化大融合
- **佛教兴盛**：「南朝四百八十寺，多少楼台烟雨中」，佛教深刻影响社会各层面

## Agent 配置

本政体包含 6 个 Agent：

| Agent | 角色 | 职责 |
|---|---|---|
| 南朝天子 | 南方协调 | 统领南方、文治方向、南方事务 |
| 北朝天子 | 北方协调 | 统领北方、武功方向、北方事务 |
| 南朝尚书 | 文治执行 | 文案策划、UI设计、文档编撰 |
| 北朝尚书 | 武功执行 | 代码开发、架构设计、系统工程 |
| 门阀 | 家族资源 | 资源调配、战略咨询、关系协调 |
| 僧统 | 佛教知识 | 知识库管理、哲理分析、文化积累 |

## 使用方式

```bash
./scripts/switch-regime.sh china/north-south
```

## 编排模式

**Federated Autonomy (联邦自治)**: 南北两个独立体系并行运作，各有天子协调、尚书执行。门阀跨越南北提供资源支持，僧统提供超越南北之上的知识与智慧。适合需要两个平行团队分工协作的场景。

## 历史特色

- **南文北武**: 文治类任务归南、技术类任务归北，发挥各自所长
- **门阀资源**: 核心资源由门阀士族掌控，重大决策须其参与
- **佛法智慧**: 僧统提供哲理层面的分析与启发
- **融合创新**: 鼓励不同技术栈与方法论的碰撞融合
