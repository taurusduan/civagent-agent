# 元朝 · 行省制 (Yuan Dynasty)

> 1271-1368 AD · Provincial System

## 简介

元朝（1271-1368）是蒙古人建立的统一王朝，疆域横跨欧亚大陆，是中国历史上版图最大的朝代。

元朝的制度特色：
- **中书省集权**：中书省是最高行政机构，总理政务，大汗直接掌控
- **行省分治**：各行省在辖区内拥有较大自主权，因地制宜，但须向中央负责
- **枢密院掌军**：军政分开但紧密协作
- **御史台监察**：独立的监察系统纠察百官
- **多元文化**：蒙古、色目、汉人、南人四等并存，多民族多文化治理

行省制度影响深远，奠定了中国此后的省级行政区划，沿用至今。

## 历史背景

成吉思汗统一蒙古草原，其后裔征服金、西夏、南宋，建立横跨欧亚的蒙古帝国。
忽必烈定国号为「元」，定都大都（北京），采用汉制与蒙古制混合的治理体系。
元朝重视商业贸易，开通海上丝绸之路，马可波罗的游记向西方展现了东方文明。
但民族等级制度和沉重赋税最终引发红巾军起义，元朝灭亡。

## 使用方式

```bash

```

## 编排模式

**Centralized Hierarchy (中央集权)**: 中书省集中决策，行省分治执行。
中央掌控大方向，地方有执行自主权。适合需要统一架构但多环境部署的场景。
