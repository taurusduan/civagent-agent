# 夏朝 · 分封家族制 (Xia Dynasty)

> c. 2070-1600 BC · Enfeoffment-Clan System

## 简介

夏朝（约公元前2070年-前1600年）是中国历史上第一个世袭王朝，由治水英雄大禹建立。夏朝标志着中国从原始部落联盟向早期国家形态的过渡，开创了「家天下」的政治传统。

其治理体制以血缘宗法为纽带，族长拥有最高决策权，下设司空、司徒、司马、牧正等职官，形成了中国最早的官僚体系雏形。大禹治水的实干精神成为这一体制的核心文化基因。

## Agent 配置

本政体包含 5 个 Agent：

| Agent | 角色 | 职责 |
|---|---|---|
| 大禹/夏王 | 族长 | 最高决策、任务分配、统领全局 |
| 司空 | 工程长官 | 代码开发、架构设计、基础设施 |
| 司徒 | 教化长官 | 文档编撰、知识传承、规范制定 |
| 司马 | 军事长官 | 安全审查、风险评估、应急响应 |
| 牧正 | 资源长官 | 资源调配、成本管控、运维保障 |

## 编排模式

**Centralized Hierarchy (中央集权)**: 夏王作为族长统一决策，各职官直接向夏王负责。简单高效的纵向管理结构，适合小型团队快速执行。

## 历史特色

- **治水精神**: 面对问题如治洪水，疏导为主，分而治之
- **朴素务实**: 不尚空谈，以实干为本
- **家天下**: 以血缘宗法维系组织凝聚力
